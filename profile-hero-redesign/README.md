# Profile Hero Redesign Skill Pack

A 5-skill workflow for turning a clean but generic profile or personal-brand hero section into a focused, modern, trust-building, conversion-driven layout.

## Workflow

1. **Hero Diagnosis** - Find what is weak, generic, unclear, or missing.
2. **Message Hierarchy** - Rewrite and structure the message so value is clear fast.
3. **Visual Composition** - Create the elevated layout, portrait treatment, and focal flow.
4. **CTA Conversion** - Strengthen the form, button, proof, and conversion path.
5. **Responsive Polish** - Make the final design feel premium on desktop and mobile.

## Best Use Cases

- Personal brand homepages
- Coach, speaker, author, creator, or consultant sites
- Newsletter landing pages
- Profile hero sections
- Service-provider websites
- Thought leadership pages

## Core Philosophy

Do not redesign for decoration. Redesign for clarity, trust, direction, and action.

The target result should feel:

- focused
- modern
- premium
- credible
- personal
- conversion-driven
- calm but confident

## Recommended Usage

Run these skills in order. Each skill produces inputs for the next skill.
