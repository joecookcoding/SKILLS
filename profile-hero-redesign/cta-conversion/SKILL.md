# Skill: CTA Conversion

## Purpose

Strengthen the call-to-action, email capture, proof placement, and conversion path in a profile or personal-brand hero section.

This skill makes the hero not only look better, but work better.

## Primary Job

Turn attention into action by making the CTA obvious, low-friction, credible, and connected to the value proposition.

## When To Use

Use this skill after the visual composition plan is complete.

It is especially useful when:

- the CTA button blends into the design
- the form feels disconnected or plain
- the offer is not compelling enough
- there is no social proof near the CTA
- the visitor has no clear next step
- multiple CTAs compete with each other
- the conversion element does not feel designed

## Required Inputs

- Final hero copy
- Desired action
- Lead magnet or offer, if any
- Form fields needed
- Available proof
- Visual composition plan
- Brand tone

## CTA Rules

Use these principles:

- One primary CTA above the fold.
- Do not ask for more form fields than needed.
- Button copy should match the value, not just the action.
- Proof should sit close to the form or button.
- Use microcopy to remove hesitation.
- Avoid equal-weight competing CTAs.
- Make the CTA large enough for mobile touch targets.

## CTA Copy Examples

For newsletters:

- Join the Newsletter
- Get the Weekly Notes
- Subscribe for Free
- Join 10K+ Readers
- Get the Next Issue

For lead magnets:

- Download the Free Guide
- Get the Checklist
- Send Me the Framework
- Get Instant Access

For coaching/consulting:

- Book a Clarity Call
- Start the Conversation
- Apply to Work Together
- See Coaching Options

## Quality Checklist

Before completing this skill, confirm:

- [ ] There is one obvious primary CTA.
- [ ] The CTA copy is specific.
- [ ] The form is visually grouped.
- [ ] Form fields are minimal.
- [ ] Proof is near the CTA.
- [ ] Microcopy reduces friction.
- [ ] The CTA is mobile-friendly.

## Red Flags

Avoid:

- Multiple equally strong CTAs.
- Long forms in the hero.
- Vague button text.
- Proof buried far below the fold.
- CTA styling that blends into the background.
- Tiny buttons or touch targets on mobile.
